# 📊 Task 6: Monitor Cloud Resources using AWS CloudWatch

## 📌 Objective

To monitor cloud resources and set alerts using AWS CloudWatch.

---

## 🛠️ Steps Performed

* Monitored EC2 instance metrics (CPU utilization)
* Created CloudWatch alarm for high CPU usage (>70%)
* Configured alerting mechanism
* Visualized metrics using CloudWatch dashboard

---

## 📊 Monitoring Setup

* Metric: CPUUtilization
* Threshold: CPU > 70%
* Evaluation Period: 5 minutes

---


## 🚀 Key Learnings

* Monitoring cloud performance
* Setting up alerts and alarms
* Importance of observability in cloud systems
