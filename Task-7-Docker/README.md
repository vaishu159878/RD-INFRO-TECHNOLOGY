# 🐳 Task 7: Dockerize an Application

## 📌 Objective

To containerize an application using Docker for easy deployment.

---

## 🛠️ Steps Performed

* Created a simple web application
* Wrote Dockerfile using Nginx base image
* Built Docker image
* Ran container and exposed port
* Verified application in browser

---

## 💻 Commands Used

```
docker build -t myapp .
docker run -p 8080:80 myapp
```

---

## 📂 Files

* Dockerfile
* index.html

---


## 🚀 Key Learnings

* Containerization using Docker
* Image vs Container concept
* Simplified application deployment
